# Assets Readme

## Required Assets

This folder should contain the following files for the game to work with proper graphics and sound:

### Images

#### Game Assets
- **bane.png** - The race track/course background
- **bil.png** - The car sprite (recommended size: 40x60 pixels)
- **katt.png** - The cat sprite (recommended size: 30x30 pixels)
- **hund.png** - The dog sprite/obstacle (recommended size: 40x40 pixels)

#### Shop Items
- **vannmelon.png** - Watermelon (Price: 50 kr)
- **blabaer.png** - Blueberries (Price: 10 kr)
- **jordbaer.png** - Strawberries (Price: 50 kr)
- **bringebaer.png** - Raspberries (Price: 20 kr)
- **kirsebaer.png** - Cherries (Price: 30 kr)
- **tyttebaer.png** - Lingonberries (Price: 10 kr)
- **bjornebaer.png** - Blackberries (Price: 25 kr)

### Audio Files

- **music.mp3** - 8-bit style background music (looping)
- **collision.mp3** - Sound effect for dog collision
- **purchase.mp3** - Sound effect for purchasing items in shop
- **win.mp3** - Sound effect for winning the race

## Fallback Graphics

**Don't have assets yet?** No problem! The game includes automatic fallback graphics:

- Track: Green grass with gray road
- Car: Red rectangle with blue window
- Cat: Orange circle with eyes
- Dog: Brown circle with eyes
- Shop items: Colored circles

## Creating Your Own Assets

### Image Recommendations

1. **PNG format** with transparency
2. **Small file sizes** for fast loading
3. **Consistent style** - consider pixel art or simple cartoon style
4. **Bright colors** for good visibility

### Audio Recommendations

1. **MP3 format** for broad browser support
2. **Short clips** (1-3 seconds for effects)
3. **8-bit/chiptune style** music for retro feel
4. **Moderate volume** - will be adjusted in code
